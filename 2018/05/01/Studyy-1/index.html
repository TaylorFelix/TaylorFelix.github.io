XHTML和HTML的区别

  • HTML是一种基本的WEB网页设计语言，XHTML的一个基于XML的置标语言
  • XHTML元素必须被正确地嵌套
  • XHTML元素必须被关闭
  • 标签名必须小写字母
  • XHTML必须有根元素

前端页面的构成

  • 结构层HTML，表示层CSS，行为层js

HTML5新特性

  • HTML已不是SGML的子集，主要是关于图像，位置，存储，多任务等功能的增加。
  • 绘画canvas
  • 用于媒介回放的video和audio元素
  • 本地离线存储localStorage长期存储数据，浏览器关闭后数据不丢失
  • sessionStorage的数据在浏览器关闭后自动删除
  • 语义化更好的内容元素，如article,footer,header,nav,section
  • 表单控件，calender,date,time,email,url,search
  • 新的技术webworker,websockt,Geolocation

移除的元素

  • 纯表现的元素,basefont,big,center,foot,s,strike,tt,u;
  • 对可用性产生负面影响的元素frame,frameset,noframes

支持HTML5新标签

  • IE8/7/6支持通过document,createElement方法产生的标签
  • 利用此特性让浏览器支持HTML5标签
  • 浏览器支持新标签后，还需要添加标签默认的样式

cookies,sessionStorage和localStorage区别

  • cookies在浏览器和服务器间来回传递，sessionStorage和localStorage不会
  • sessionStorage和localStorage的存储空间更大
  • sessionStorage和localStorage有更多丰富易用的接口
  • sessionStorage和localStorage各自独立的存储空间

浏览器内多个标签之间的通信

  • 调用localStorage,cookies等本地存储方式

CSS元素分类

  • 块级元素 div,p,h1,form,ul,li
  • 行内元素 span,a,label,input,img,strong,em;

CSS隐藏元素的方法

  • Opacity元素本身依旧占用位置并起作用，起响应用户交互
  • Visibility与opacity唯一不同的是它不会响应任何用户交互
  • Display设置None让交互操作失去作用
  • Position不会影响布局，能让元素保持可以操作
  • Clip-path

CSS清除浮动的几种方法

  • 使用带clear属性的空元素
  • 使用CSS的overflow属性
  • 使用CSS的after伪元素
  • 使用邻接元素处理

CSS居中

内联元素居中

  • 水平居中

    行内元素：text-align:center
    Flex布局：设置display:flex;justify-content:center

  • 垂直居中设置

    父元素高度确定的单行文本：设置height=line-height
    父元素高度确定的多行文本：插入table或先设置display:table-cell，设置vertical-align:middle

块级元素居中

  • 水平居中：左右margin:auto
  • 不定宽块状元素

    在元素外加入table标签，元素写在td内，设置margin:auto
    给元素设置display:inine方法
    父元素设置position:relative和left:50%，子元素设置position:relative和left:50%

  • 垂直居中设置

    使用position:absolute，设置left,top,margin-left等
    利用position:fixed属性
    使用css3新属性transform:translate属性
    使用:before元素

页面导入样式时，使用link和@import的区别

  • link属于XHTML标签，除了加载CSS外，还能定义RSS，定义rel连接属性等作用，而@import是CSS提供的，只能用于加载CSS
  • 页面被加载时，link会同步加载，而@import会等待页面被加载完再加载
  • import是CSS2.1提出的，IE5以上才能识别，而link是XHTML标签，无兼容问题

CSS盒子模型

  • IE盒子模型，标准W3C盒子模型，IE的content部分包含了border和padding
  • 盒模型：content,padding,margin,border

CSS选择符，可继承，优先级算法

  • id选择器(#myid)
  • 类选择器(.myclassname)
  • 标签选择器(div,h1,p)
  • 相邻选择器(h1+p)
  • 子选择器(ul>li)
  • 后代选择器(li a)
  • 通配符选择器*
  • 属性选择器(a[rel=”external”])
  • 伪类选择器(a:hover,li:nth-child)
  • 可继承的样式(foot-size,foot-family,color,ul,li,dl,dd,dt)
  • 不可继承的样式(border,padding,margin,width,height)
  • 优先级就近原则，同权重情况下样式定义最近者为准
  • !import > id > class > tag
  • import比内联优先级高

CSS3新特性

  • CSS3实现圆角(border-radius:8px),阴影(box-shadow:10px),文字特效(text-shadow),线性渐变(gradient),旋转(transform)

javascript的typeof返回数据类型

  • Object number function boolean underfind

强制类型转换和隐式类型转换

  • 强制(perseInt,parseFloat,numer)，隐式==-===

数组方法

  • push()尾部添加，pop()尾部删除
  • Unshift()头部添加，shift()头部删除
